and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.1] - 2025-12-07

### Added

## [0.1.0] - 2025-12-07

### Added

- Initial release
- Tree view sidebar displaying epics and stories from `sprint-status.yaml`
- Workflow actions with inline play button for actionable stories
- Status management via right-click context menu
- Sprint file selection for workspaces with multiple `sprint-status.yaml` files
- Terminal integration for running Claude workflow commands
- Auto-refresh when `sprint-status.yaml` changes
- Support for story statuses: backlog, ready-for-dev, in-progress, review, done
- Workflow commands for create-story, dev-story, and code-review
